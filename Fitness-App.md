# Fitness Club React App
A modern fitness club web application built with React that allows users to explore exercises, view details, and navigate through workout categories using a clean and interactive UI

## **Tech Stack Used**
- React (v18)
- React Router DOM
- Material UI (MUI)
- Emotion (styled & react)
- React Horizontal Scrolling Menu
- React Loader Spinner
- JavaScript (ES6+)
- Custom CSS
- Vite

## **Key Features**
- Multi-page navigation using React Router  
- Browse exercises by category  
- Horizontal scrolling for exercise categories  
- Exercise detail pages with rich UI  
- Loading spinner while fetching or processing data  
- Responsive design for mobile and desktop  
- Modern UI using Material UI components and icons  
- Smooth user experience with reusable components
- The motion of each exercise image is achieved using **animated GIFs**

## **Why These Technologies**
- **React:** Component-based architecture and state management  
- **React Router:** Smooth SPA navigation without reloads  
- **Material UI:** Prebuilt, accessible, and modern UI components  
- **Emotion:** CSS-in-JS styling with theme support  
- **Horizontal Scroll Library:** Better UX for browsing categories  
- **Vite:** Fast development and optimized builds  

## **Challenges & Solutions**
- **Handling complex UI layouts:**  
  Solved using Material UI components and grid system  
- **Managing navigation between pages:**  
  Implemented route-based navigation with React Router  
- **Improving user experience during loading:**  
  Added loader spinner for better feedback  
- **Responsive design:**  
  Achieved using MUI breakpoints and flexible layouts
