# Directory App 

## **Tech Stack Used**
- React  
- Vite  
- JavaScript (ES6+)  
- Custom CSS  
- LocalStorage  
- Firebase 

---

## **One-Line Summary**
- A simple and responsive directory app built with React to add, search, and manage entries with LocalStorage persistence.

---

## **Key Features**
- Add new directory entries  
- Search/filter entries by name or Aadhaar  
- Display all records dynamically  
- Prevent duplicate entries  
- Delete a single entry  
- Clear all entries  
- Data saved in LocalStorage  
- Fully responsive design
