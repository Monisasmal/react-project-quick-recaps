# Find Location App

Find Location is a React application that integrates Leaflet maps using React Leaflet.  
I implemented interactive maps with markers and zoom controls, managed map rendering using React Hooks, and styled the UI with custom CSS.  
The project demonstrates my ability to integrate third-party libraries and work with geolocation-based UIs.

## **Tech Stack Used**
- React (v19)
- React Leaflet
- Leaflet
- Vite
- JavaScript (ES6+)
- React Hooks
- Custom CSS

---

## **Key Features**
- Interactive map rendering using Leaflet
- Display map tiles and zoom controls
- Marker placement to highlight locations
- Dynamic map updates based on user interaction
- Responsive map layout
- Clean and minimal UI
- Fast loading and smooth interaction

## **How Map Functionality Works**
- The map is rendered using `MapContainer`
- Tile layers load map visuals from a provider
- Markers are placed at specified coordinates
- Users can zoom and pan the map
- React manages re-rendering when state or props change
